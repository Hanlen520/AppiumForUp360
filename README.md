# AppiumUp360自动化测试框架
## 项目名及简介:
此项目是基于appium和Selenium开源工具封装而成的自动化app测试工具

## 该框架所需python插件:
* pyyaml
* xlrd
* xlutils
* configparser
* HTMLTestRunner
* Appium-Python-Client

## 框架介绍:
* unittest参数化
* PageObject分层管理
* 本地化多机日志管理
* 用例编写基于yaml配置
* 支持多设备andoird并行
* 支持失败重连
* 自动生成HTML测试报告

## 命令运行:
* python  parents_run.py  # 执行家长端测试脚本
* python  teacher_run.py  # 执行老师端测试脚本
